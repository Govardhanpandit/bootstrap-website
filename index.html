The MouseEvent Object handles events that occur when the mouse interacts with the HTML document.

# Event Occurs When
1 onclick: A user clicks on an element
2 oncontextmenu: A user right-clicks on an element
3 ondblclick: A user double-clicks on an element
4 onmousedown: A mouse button is pressed over an element
5 onmouseenter: The mouse pointer moves into an element
6 onmouseleave: The mouse pointer moves out of an element
7 onmousemove: The mouse pointer moves over an element
8 onmouseout: The mouse pointer moves out of an element
9 onmouseover: The mouse pointer moves onto an element
10 onmouseup: A mouse button is released over an element
